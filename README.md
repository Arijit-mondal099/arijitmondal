# My Portfolio

A modern, high-performance developer portfolio built with the latest web technologies. This project showcases my skills, projects, and professional experience in a clean, responsive, and interactive interface.

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/) & [Tailwind Animate](https://github.com/jamiebuilds/tailwindcss-animate)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) & [Lucide React](https://lucide.dev/)
- **Form Validation:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Language:** TypeScript
